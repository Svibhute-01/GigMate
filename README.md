# GigMate 🚀

**GigMate** is a freelance marketplace platform that connects clients with skilled freelancers. Users can create gigs, explore services, manage projects, and collaborate efficiently through a modern and responsive web application.

---

## ✨ Features

* 👤 User Authentication & Authorization
* 📝 Create, Edit, and Manage Gigs
* 📱 Responsive Design for All Devices
* 🔒 Secure User Data Management
* 📊 Dashboard for Users and Freelancers

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

* Node.js
* MongoDB
* Git

### Installation

1. Clone the repository

```bash
git clone https://github.com/Svibhute-01/GigMate.git
cd GigMate
```

2. Install dependencies

```bash
npm install
```

3. Configure environment variables

Create a `.env` file and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

4. Run the application

```bash
npm start
```

For development:

```bash
npm run dev
```
